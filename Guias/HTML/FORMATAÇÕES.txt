Formatações 

<em> efatiza um texto

<mark> destaca um texto

<del> risca o texto

<q> usado para citações simples("")

<blockquote> usado para citações completas(tem quebra de linha e um espaço)

<small> deixa o texto pequeno

<sup> X^20 <p>x<sup>20</sup></p>

<sub> H2O <p>H<sub>2</sub>O</p>

<abbr> coloca o significado das abreviações

<hr> criar uma divisão entre dois elementos

<pre> O texto é mostrado como no código

<code> mostra um pedaço de código no site

<pre>
        <code>
            "img src="midias/logo-html.png" alt="logo-html"
        </code>
</pre>


&lt; &gt; serve para colocar <>
